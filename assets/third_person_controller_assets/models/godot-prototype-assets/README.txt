# Godot Prototype Kit

A modular block out kit for Godot 4.x

## What's Included
- 4 wall pieces
- 2 floor/ceiling tiles
- 10 props (boxes, stairs, ramp, pillar, table, barrier, platform)

## How To Use
1. Drop the prototype kit folder into your Godot project
2. Instance any scene from scenes/walls or scenes/props
3. Snap pieces together on a 4x4 meter grid
